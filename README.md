# 🧮 Calculator

A modern, fully-functional web calculator with dynamic theme switching and a sleek user interface.

## ✨ Features

- **Basic Arithmetic Operations**: Addition, subtraction, multiplication, division, and percentage calculations
- **Decimal Support**: Full support for decimal numbers with comma separator (locale-aware)
- **Smart Input Validation**: Prevents invalid operator sequences and handles edge cases gracefully
- **Dark/Light Theme Switching**: Toggle between dark and light themes with smooth transitions
- **Persistent Preferences**: Your theme choice is saved and restored on next visit using localStorage
- **Responsive Design**: Optimized for both desktop and mobile devices
- **Interactive Feedback**: Hover effects and button animations for enhanced user experience
- **Error Handling**: Gracefully handles invalid expressions and mathematical errors

## 🎨 Design Highlights

### Dark Theme (Default)

- Black background (#000000)
- Orange operators (#FF9F0A)
- Gray number buttons (#333333)
- High contrast white text

### Light Theme

- Clean white background (#ffffff)
- Orange operators with light background
- Light gray number buttons
- High contrast dark text
